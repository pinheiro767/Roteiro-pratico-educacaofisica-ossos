Guia de Estudo: Sistema Esquelético

Curso: Educação Física

Instituição: Universidade Estadual de Maringá - UEM

Professora: Profª Dra. Cláudia Pinheiro

Material de apoio para revisão, estudo dirigido e registro de anotações pelos alunos.

Objetivo Revisar os principais acidentes anatômicos e estruturas ósseas do sistema esquelético.
Uso do material Preencha seus dados, faça anotações e use o botão de salvar para manter tudo gravado no navegador.
Exportação Use o botão “Imprimir / Salvar em PDF” para gerar uma versão em PDF com suas anotações.

Identificação do aluno

Dica: ao abrir pelo GitHub Pages, o app poderá ser instalado no celular e usado com cache offline.

Conceitos iniciais

1. Defina esqueleto apendicular e esqueleto axial.

2. Explique o que é diáfise, epífise, metáfise, cartilagem epifisial e linha epifisial.
